Median Height


Educational game that teaches "Sorting" concept

ABOUT MEDIAN HEIGHT

Sorting is a process of arranging given items in a systematic way The purpose of this project is to develop a simple web-based game that teaches the Sorting concept in a fun and engaging way.

  • Target users: Primary and secondary school children
  • Timeframe:December 2018 - April 2019

CHALLENGES

  • Teach the sorting concept to pre-school and secondary school children in a simple and fun way.
  • Create a simple game interface with minimum graphics that is easy to understand , fun and engaging to play.

MY ROLE

My role in this project is UX/UI and a python developer. I conducted user research, conceptualized the user insights, created user flows, then developed wireframes and User interface for this game. After creating the user interface, I conducted usability testing with three members for validation and made the required changes analyzed from the feedback from my supervisors and usability test results.

DESIGN AND DEVELOPMENT KIT

DESIGN PROCESS

User research:I have conducted user research by volunteering in science events at Maynooth University, Ireland, where children play games similar to the delivery truck game. By helping children while playing similar games, I took brief notes on the challenges they encountered while playing and received feedback from the children and their parents about their experiences playing the games.

Conceptualization and User flow: From the research insights and feedback from the children, I came up with the concept of how to develop a simple web-based game that teaches the Hamiltonian path concept as follows: A certain number of people representing their heights are displayed. The heights are represented with a set of natural numbers. The heights are represented in feet using single digit numbers or a double-digit number: 1)The minimum length of the list should be greater than or equal to three. 2)The number of natural numbers should always be of odd. 3)All the numbers should be unique. The numbers displayed at random. The goal of the game is to nd the median number of the given list of numbers.

User Interface design and development :
I edited the game avatars using Inkscape and developed the game using python. After creating the prototype, I conducted the Usability testing using three users. I took brief notes during the tests and also feedback from my supervisor and applied the required changes to the project for better usability.
